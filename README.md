# Mini WhatsApp

## Description
Real-time chat application with chat rooms and messaging.

## Tech Stack
- Node.js
- Express.js
- EJS
- MongoDB
- JavaScript

## Features
- Real-time chat
- Multiple chat rooms
- Interactive UI

## How to Run
1. Clone the repository
2. Install dependencies with `npm install`
3. Start the server with `nodemon index.js`
4. Open `http://localhost:8080` in your browser
